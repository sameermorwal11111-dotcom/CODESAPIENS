RED BOX vs GREEN BOX - VISUAL GUIDE
════════════════════════════════════════════════════════════════════════════════

GREEN BOX:
  Meaning: Face recognized
  Confidence: < 70%
  Label: "JOHN (92%)" or "SARAH (65%)"
  Thickness: Thick border
  Action: Match counted
  Color: RGB(0, 255, 0)

RED BOX:
  Meaning: Face NOT recognized / Unknown user
  Confidence: >= 70%
  Label: "UNKNOWN (71%)" or "UNKNOWN (95%)"
  Thickness: Regular border
  Action: No match counted
  Color: RGB(0, 0, 255)

EXAMPLE SCENARIO:
  Camera shows 2 faces:
    Face 1: GREEN box "JOHN (92%)"        ← Recognized
    Face 2: RED box "UNKNOWN (72%)"       ← NOT recognized (no match)
  
  Results: john: 15 matches | unknown: 0 matches

ACCURACY IMPROVEMENTS EXPLAINED
════════════════════════════════════════════════════════════════════════════════

This version is MORE ACCURATE because:

1. HISTOGRAM EQUALIZATION
   - Makes faces look clearer
   - Better contrast and brightness
   - Applied during both training AND recognition

2. CONFIDENCE AVERAGING
   - Looks at last 30 frames
   - Takes average confidence
   - Reduces false positives/negatives
   - Smoother results

3. ENHANCED LBPH PARAMETERS
   - radius=1 (more detail)
   - neighbors=8 (more precision)
   - grid_x=8, grid_y=8 (better spatial analysis)

4. STRICTER FACE DETECTION
   - minNeighbors=6 (was 5, now more strict)
   - minSize=(80, 80) (better quality faces)
   - maxSize=(400, 400) (not too close)

5. 100 TRAINING IMAGES
   - More angles captured
   - More lighting variations
   - Better generalization
   - Reduces recognition errors

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

PROBLEM: Webcam not opening
SOLUTION:
  1. Close other apps using camera (Zoom, Teams, Discord)
  2. Restart computer
  3. Check Windows Settings → Privacy & security → Camera
  4. Try port 1 instead of 0 (uncommented in Detector.py if needed)

PROBLEM: Poor recognition accuracy
SOLUTION:
  1. Ensure good lighting during capture
  2. Capture at 100 images (not 50)
  3. Stand at different angles
  4. Different distances from camera
  5. Retrain the model

PROBLEM: Too many unknown faces
SOLUTION:
  1. This is CORRECT behavior for unknown people
  2. Only trained users get recognized
  3. Others get RED box + "UNKNOWN"
  4. Add more trained users for more recognition

PROBLEM: Recognition too slow
SOLUTION:
  1. Use Option 3, Press 1 for 10 seconds (Quick)
  2. Fewer trained users = faster processing
  3. Larger face = faster detection
