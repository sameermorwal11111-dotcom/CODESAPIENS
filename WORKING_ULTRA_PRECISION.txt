======================================================================
TIPS FOR BEST RESULTS
======================================================================

During Capture:
  - Stand 30-50cm from camera (optimal distance)
  - Good lighting (natural light is best)
  - Face centered
  - Different angles
  - Relaxed face (natural look)
  - Let system capture all 100 images

During Training:
  - Just wait (system does everything)
  - Don't interrupt
  - Check for "TRAINING COMPLETE" message

During Recognition:
  - Same distance: 30-50cm
  - Same lighting conditions
  - Face clearly visible
  - Use 15-30 seconds for best results
  - Don't move too fast

======================================================================
TROUBLESHOOTING
======================================================================

Problem: Script won't start
Solution:
  1. Make sure you're in the right directory
  2. Type the full path exactly as shown above
  3. Check for typos

Problem: Camera won't open
Solution:
  1. Close other apps using camera (Zoom, Teams)
  2. Restart computer
  3. Check Windows camera permissions
  4. Try plugging/unplugging camera

Problem: Poor accuracy
Solution:
  1. Use good lighting during capture
  2. Stand at correct distance (30-50cm)
  3. Capture all 100 images
  4. Train with all images
  5. Retrain if needed

Problem: Recognition says UNKNOWN
Solution:
  1. Normal for untrained people
  2. Add more trained users with Option 1+2
  3. Or train the person again with more images
